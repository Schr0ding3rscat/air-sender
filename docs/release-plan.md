# 12-Month Delivery Plan

## M1-M2 Foundation

- Monorepo setup, IPC API baseline, session lifecycle skeleton
- GPU probe abstraction layer
- Compliance intake (AirPlay/Cast/Miracast)

## M3-M5 First protocol

- AirPlay mirroring bring-up across target operating systems
- Pairing, audio sync, single-session recording

## M6-M8 Android coverage

- Cast adapter MVP + Miracast adapter MVP
- Dynamic protocol routing and multi-session support (up to 4)
- Admin CLI and policy enforcement

## M9-M10 Hardening

- Compatibility matrix runs
- Reconnect and impairment resilience
- Installer/signing hardening and update channels

## M11-M12 Release readiness

- Security/performance acceptance
- Docs/runbooks and DRM feasibility report
- GA gate review
